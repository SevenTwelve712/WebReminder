'seven'

access
  • path
  • mode
  • Всякая херня
 Чекает уровень доступа у текущего юзера к объекту (директория которого path). Mode = os.F_OK (объект существует) / os.R_OK (доступен на чтение) / os.W_OK (доступен на запись) / os.X_OK (доступен на исполнение). Возвращает true / false access("C:\Program Files\Google\Chrome\Application\chrome.exe", X_OK)
True
listdir path Выдает список из объектов (папок файлов и прочей херни) в path listdir(path='C:\Program Files\Google\Chrome\Application)
['115.0.5790.173', 'chrome.exe', 'chrome.VisualElementsManifest.xml', 'chrome_proxy.exe', 'master_preferences', 'SetupMetrics']
mkdir
  • path
  • Прочая хуета
 Создает папку по пути path. Если папка есть, то OS error makedirs("../Frol huesos/1/2")
Теперь на порядок выше есть папка Frol huesos, в которой есть папка 1, в которой есть папка 2
makedirs
  • path
  • Прочая хуета
 Создает папку с промежуточными папками. Но никакие из создающихся папок не должны уже быть, иначе OS error makedirs("../Frol huesos/1/2")
Теперь на порядок выше есть папка Frol huesos, в которой есть папка 1, в которой есть папка 2
remove
  • path
  • Прочая хуета
 Удаляет путь к файлу remove('C:\test os\Frol huesos\1.txt')
Теперь в папке Frol huesos нет файла 1.txt
rename
  • a
  • b
 Переименовывает a в b rename('1.docx', '2.docx')
Теперь файл 1.docx называется 2.docx
system command Выполняет системную команду и возвращает код завершения (в случае успеха 0) system("taskmgr")
Откроет диспетчер задач, а после его закрытия вернет 0

  • path
  • start=None
 Вычисляет путь к path относительно директории start (по умолчанию директория исполняемого файла) path.relpath("C:\Users\seven\PycharmProjects\1\1.py", start="C:\Program Files")
