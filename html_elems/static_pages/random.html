randrange
  • start
  • stop=None
  • step=1
 Возвращает случайное число из последовательности random.randrange(0, 10, 2)
6
randint
  • a
  • b
 Случайное число, a <= n <= b random.randint(3, 9)
4
choice list Случайный элемент непустой последовательности random.choice([1, 2, 3, 4])
3
shuffle list Перемешивает list. Работает только для изменяемых объектов a = [1, 2, 3, 4, 5]
random.shuffle(a)
a --> [1, 5, 2, 4, 3]
sample
  • sequence
  • k
 Возвращает список длиной k из sequence random.sample([1, 2, 3, 4, 5, 6], 3)
[1, 5, 3]
