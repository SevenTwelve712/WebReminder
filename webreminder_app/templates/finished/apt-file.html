apt-file — утилита командной строки для поиска пакетов по файлам, содержащимся в пакетах apt

Синтаксис

apt-file option command target

В качестве target могут выступать, например, пакет или файл

По факту, все сводится к этим двум командам:

apt-file options search pattern apt-file options show package
