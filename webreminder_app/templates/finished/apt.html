Apt — программа для установки, апгрейда, удаления и прочей хуйни для пакетов в ос debian

sources.list

Для того, чтобы apt знал, откуда устанавливать пакеты, ему нужно указать репозитории в файле sources.list. По дефолту этот файл лежит в /etc/apt

Как записывать репозитории:

deb ссылка_на_репозиторий релиз_дебиана контент

Виды контента

  • main: основные по
  • contrilb: свободные по, но их зависимости могут быть несвободными
  • non-free: содержит несвободные пакеты
  • non-free-firmware: предоставляет несвободное по, необходимое для некоторых аппаратных средств
