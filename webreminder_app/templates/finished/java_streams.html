Базовые потоки

В java потоки делятся в основном на два типа:

  • байтовые: представлены абстрактными классами InputStream и OutputStream
  • символьные: представлены абстрактными классами Reader и Writer

Здесь мы будем рассматривать только работу с текстовыми файлами.

Файловые потоки

Представлены классами FileWriter и FileReader.

Мы не будем пользоваться этими классами для ввода/вывода информации, м будем их передавать в качестве аргументов другим потокам/объектам

Для инициализации будем пользоваться такими конструкторами:

  • FileReader(String fileName)
  • FileWriter(String fileName); можно указать второй аргумент true, тогда будет дозапись в файл

Буферизованные потоки

Буферизованные потоки не обращаются каждый раз к источнику для получения/записи информации. Вместо этого они записывают все в буфер.

Операции записи информации из буфера осуществляются при:

  • Переполнении буфера
  • Закрытии потока
  • Вызове метода потока .flush()

Буферизованный поток считывания считывает всю доступную информацию (доступную по размеру буфера) и дальше при операциях считывании берет инфу из буфера. Когда буфер заканчивается, поток вновь обращается к источнику

В данном руководстве мы рассмотрим два буферизованных потока: BufferedWriter — буферизованный поток для записи текста и BufferedReader — буферизованный поток для считывания текста.

Они имеют следующие конструкторы:

  • Buffered[Reader, Writer]([Reader, Writer] [in, out])
  • Buffered[Reader, Writer]([Reader, Writer] [in, out], int sz), где sz — размер буфера в символах, по умолчанию 8192
