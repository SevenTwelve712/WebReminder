Базовые типы данных

Для инициализации объекта базового типа данных не нужно указывать new, так как это примитив (т.е. не объект)

  • boolean
  • byte: хранит целое от -128 до 127
  • short: хранит целое от -32768 до 32767
  • int: хранит целое от -2147483648 до 2147483647
  • long: хранит целое от –9 223 372 036 854 775 808 до 9 223 372 036 854 775 807. При присвоении значения большим числам необходимо добавлять L в конец: long num = 2147483649L (т.к. по умолчанию целочисленное значение это int и не может быть больше 2147483647)
  • double: хранит дробное от ±4.9*10-324 до ±1.7976931348623157*10308
  • float: хранит дробное от -3.4*1038 до 3.4*1038. При назначении тот же прикол, что и с long; java все дробные рассматривает как double, поэтому говорим ему, что это float: float fl = 30.6F;
  • char: хранит одиночный символ в кодировке utf-16

String

String немного отличается. Он не является базовым типом, но при его инициализации не надо указывать слово new. Java использует string pool для хранения одинаковых строк. Если строка уже есть в пуле, то Java не создает объект, а дает ссылку на существующий. String s = new String("hello"); создаст новый объект в памяти, не в пуле, и не будет равен String s1 = String("hello");

Операторы

Численные

  • +: сложение
  • -: вычитание
  • *: умножение
  • /: деление (учитывайте, что int / int > float)
  • %: остаток
  • ++: увеличивает значение переменной на единицу. Если до переменной, то сначала увеличивается, потом присваивается, если после, то наоборот
  • --: аналогично ++, только уменьшает на 1

Логические и условные

  • ==: равенство
  • !=: неравенство
  • <: меньше
  • >: больше
  • <=: меньше или равно
  • >=: больше или равно
  • |: или
  • &: и
  • !: не
  • ^: исключающее или
  • ||: или, но если первое True, то второе не проверяется
  • &&: и, но если первое False, то второе не проверяется

Присваивания

  • =: присваивание
  • a sign=b: a присваивается a sign b
