Запуск скриптов

особенность jav`ы состоит в том, что ее скрипты сначала компилируются в байт-код, а затем выполняются специальной машиной (Java Virrtual Machine, JVM). Это позволяет запускать код на разных платформах, протсо нужно под каждую платформу написать работающий JVM

Получается, для работы скрипта нужны:

  • Компилятор, например, javac
  • Сама JVM

Эти два компонента объединены в JDK: java Development Kit. Существуют две популярные реализации: свободная и от Oracle.

JDK от oracle можно скачать здесь (для windows)

После установки jdk установится и компилятор, и jvm. Как теперь запустить скрипт Program.java:

  1. Скомпилировать его: javac Program.java
  2. После этого создастся файл *.class с байт-кодом, нам надо запустить его: java класс который мы хотим запустить

Ввод и вывод

Для вывода используем System.out.println("some text")

Для вывода воспользуемся иным классом; импортируем его: import java.util.Scanner;

Затем определим его объект: Scanner in = new Scanner(System.in);

И будем пользоваться его методами: int num = in.nextInt();

NB: nextInt считывает следующее число, но не считывает символ перевода строки. nextLine после nextInt будет пустым
