Где писать и подключать стили

Css — язык разметки, определяющий стили в документах html. Обычно стили прописываются в отдельном документе с расширением .css. Для того чтобы использовать стили из .css файла, его нужно подключить в <head></head>

{{ code_block('<link rel="stylesheet" type="text/css" href="ссылка на файл">') }}

Конечно, можно еще прописывать стили в самом .html файле (это как то делается в <head></head>) или в атрибуте style тега:

{{ code_block('<div style="margin-top: 20px;">Этот div отстоит от границ верхнего элемента на 20 пикселей</div>') }}

но лучше это делать в отдельном документе.

Как писать стили в .css

Общий вид:

{{ code_block(css_info) }}

Как можно задать селектор:

  • Через упоминание тега. Таким образом, мы зададим стили для всех тегов в документах, к которым подключим файл:
    • {{ code_block(tag) }}

    Теперь у всех заголовков 2 уровня будет шрифт for_captions 35 кегля

  • Через классы. Для этого надо задать имя класса, например, .my_class, а затем в тегах, в которых хотим использовать стиль my_class, указать его в атрибуте class:
    • {{ code_block(class) }} {{ code_block('<div class="my_class">У этого div`a будет сплошной черный бордер шириной 3 пикселя</div>') }}
