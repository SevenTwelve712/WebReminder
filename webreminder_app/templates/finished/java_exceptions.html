Исключения

В java есть два класса исключений, наследующихся от класса Throwable:

  • Error: внутренние ошибки в java, на них мы почти не можем повлиять
  • Exception: сами исключения, который бывают:
    1. непроверяемыми: наследуются от RuntimeException, не обрабатываются с помощью try catch
    2. проверяемые: все остальные

throw и throws

С помощью этого оператора мы можем выбросить исключение:

{{ code_block(throw) }}

Однако есть нюанс: в java все выброшенные исключения должны быть обработаны. Поэтому если мы выбрасываем исключение, то мы его должны либо сразу обработать через catch, либо же в методе, в котором может быть выброшено исключение указать: public void method(Object arg) throws SomeException {}

Тогда при использовании этого метода мы будем обязаны либо пробросить исключение дальше, либо обработать его. Таким образом гарантируется, что все исключения будут обработаны.
