Установка компилятора и запуск программы

Мы будем пользоваться компилятором fortran от intel. (ifort или его более новая версия ifx). Установим его:

  1. Перейдите по этой ссылке и установите компилятор, выбрав нужные параметры (скорее всего вам необходим компилятор для windows, offline версии.
    Для того, чтобы скачать компилятор, нажмите Continue as a Guest (download starts immediately) →
    Либо вы можете скачать оффлайн установщик для windows прямо с этого сервера по этой ссылке
  2. Вы скачаете файл. Это программа установки компилятора. Вам нужно запустить ее. Программа предложит распаковать временные файлы, можете распаковать их в любую директорию, они все равно потом удалятся. Затем начнется сам процесс инсталляции. Выберите установку по умолчанию, если не хотите париться насчет установки, или выберите компоненты (там можно убрать интеграцию с visual studio, запретить передачу информации о нас и прочее)
  3. После того, как установка пройдет, можете закрыть окно. Теперь в ваших приложениях появится что то вроде Intel oneAPI command prompt for Intel 64. Откройте его. Это особая версия командной строки, выглядит примерно так:
  4. Затем перейдите в вашу директорию с проектом командами cd. Например, в моем случае команда перехода в рабочую директорию выглядела бы так: {{ code_block("cd C:\\Users\\seven\\Documents\\Vagray\\my_work") }}
  5. Для компиляции файла (создание .exe) выполните (введите в командную строку и нажмите enter)
    6. {{ code_block("ifort ваш_скрипт.f -o создаваемый_exe_файл.exe") }}
  6. Это создаст .exe в вашей рабочей директории, можете запускать его. Это и будет исполняемый файл вашей программы

Примечания

  • Вы в принципе можете не переходить в рабочую директорию, а указать абсолютное имя скрипта и абсолютное имя программы. Например, в моем случае следующая команда сработала бы, если бы я не менял директорию: {{ code_block("ifort C:\\Users\\seven\\Documents\\Vagray\\my_work\\vagin_changed_formats.f
    -o C:\\Users\\seven\\result.exe") }} Она создала бы исполняемый result.exe в моей директории пользователя (C:\Users\seven)
  • Вы можете использовать не компилятор ifort, а его более новую версию: ifx. Для этого в команде компиляции замените ifort на ifx {{ code_block("ifx ваш_скрипт.f -o создаваемый_exe_файл.exe") }}
