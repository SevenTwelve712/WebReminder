Стандарт javadoc

В java есть стандарт документации к классам/методам. Она пишется до класса, начинается с /** и заканчивается */. Между этими символами строки комментариев начинаются с *:

{{ code_block(comment) }}

В строках комментария можно указать теги, начинающиеся с @ и представляющие определенную информацию

  • @param: документация к параметру
  • @returns: что возвращает
  • @throws: что может выбрасывать как исключение
